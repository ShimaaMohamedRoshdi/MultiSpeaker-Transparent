# لوحة متابعة جلسة المحكمة

واجهة أمامية احترافية تحاكي لوحة متابعة محضر جلسة المحكمة بشكل فوري. مبنية باستخدام React وTypeScript وVite وZustand وTailwind CSS، مع بث لحظي تجريبي (بدون ميكروفونات أو خدمات تحويل كلام إلى نص).

## نظرة عامة

يوفر التطبيق واجهة متابعة مهنية للجلسة مع أدوار ثابتة للمتحدثين، تحكم في كتم الأدوار، تعديل فوري للعبارات، وإمكانية الطباعة/التصدير. تصل الرسائل كجمل كاملة فقط لمحاكاة محضر جلسة حي.

## أهم المميزات

- أدوار خاصة بالمحكمة مع شارات عربية وألوان وأيقونات فريدة
- محاكاة بث لحظي عبر WebSocket كل 1–3 ثوانٍ
- تعديل داخل الرسالة مع تحديث فوري لحالة Zustand
- كتم لكل دور (الرسائل المكتومة لا تظهر في المحضر)
- إيقاف/استئناف البث، محاكاة إعادة الاتصال، ومؤشر حالة مباشر
- بحث في المحضر مع تمرير تلقائي لآخر رسالة
- طباعة المحضر وتصدير البيانات بصيغة JSON

## آلية المحاكاة الفورية

تُصدر خدمة البث التجريبية رسائل كاملة كل 1–3 ثوانٍ عبر مؤقت. كل رسالة تحتوي على المعرف، بيانات المتحدث، الدور، الطابع الزمني، وحالات الكتم/التعديل. لا يتم إرسال مقاطع جزئية.

## بنية Zustand

- `transcripts`: قائمة رسائل المحضر
- `mutedRoles`: حالات كتم الأدوار
- `connectionStatus`: متصل، غير متصل، إعادة اتصال
- `isPaused`: إيقاف استقبال الرسائل دون قطع الاتصال
- `searchQuery`: نص البحث الحالي

## منطق الكتم

- لكل دور زر كتم مستقل
- عند الكتم يتم إسقاط الرسائل الجديدة لهذا الدور مباشرة
- عند إلغاء الكتم تستمر الرسائل القادمة دون استرجاع السابقة

## سير التعديل

- يمكن تعديل أي رسالة داخل المحضر
- التعديل يحدث Zustand فوراً ويضبط `isEdited = true`
- أي تحديثات لاحقة لن تستبدل النص المعدل

## الطباعة والتصدير

- زر "طباعة الجلسة" يفتح عرضا جاهزا للطباعة عبر `window.print()`
- التصدير بصيغة JSON لحفظ سجلات الجلسة

## التشغيل
